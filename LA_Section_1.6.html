6.

I’ll list the elements in the order they appear from left to right in the equation:

\(\begin{bmatrix} \text{Na}\\ \text{P}\\ \text{O}\\ \text{Ba}\\ \text{N}\\ \end{bmatrix}\)

So, the first molecule is: \(\begin{bmatrix} 3\\ 1\\ 4\\ 0\\ 0 \end{bmatrix}\)

The vector equation we want to solve is then:

\(x_1\begin{bmatrix} 3\\ 1\\ 4\\ 0\\ 0 \end{bmatrix}+x_2\begin{bmatrix}0\\0\\6\\1\\2\end{bmatrix}=x_3\begin{bmatrix}0\\2\\8\\3\\0\end{bmatrix}+x_4\begin{bmatrix}1\\0\\3\\0\\1\end{bmatrix}\).

Make this a homogeneous vector equation:

\(x_1\begin{bmatrix} 3\\ 1\\ 4\\ 0\\ 0 \end{bmatrix}+x_2\begin{bmatrix}0\\0\\6\\1\\2\end{bmatrix}-x_3\begin{bmatrix}0\\2\\8\\3\\0\end{bmatrix}-x_4\begin{bmatrix}1\\0\\3\\0\\1\end{bmatrix}=\begin{bmatrix}0\\0\\0\\0\\0\end{bmatrix}\).

The corresponding coefficient matrix, \(A\), is:

\(\begin{bmatrix}3&0&0&-1\\ 1&0&-2&0\\ 4&6&-8&-3\\ 0&1&-3&0\\ 0&2&0&-1 \end{bmatrix}\)

which reduces to

\(\begin{bmatrix} 1&0&0&-1/3\\ 0&1&0&-1/2\\ 0&0&1&-1/6\\ 0&0&0&0\\ 0&0&0&0\\ \end{bmatrix}\)

Read of the equations this matrix represents:

\(x_1-\frac{1}{3}x_4=0\),

\(x_2-\frac{1}{2}x_4=0\), and

\(x_3-\frac{1}{6}x_4=0\).

The solution to the system is:

\({\bf x}=x_4\begin{bmatrix}\frac {1}{3}\\ \\\frac{1}{2}\\ \\ \frac{1}{6}\\ \\1\end{bmatrix}\)

Pick a nice value for \(x_4\) to balance the equation: \(x_4=6\), so :

\({\bf x}=6\begin{bmatrix}\frac {1}{3}\\ \\\frac{1}{2}\\ \\ \frac{1}{6}\\ \\1\end{bmatrix}=\begin{bmatrix}2\\3\\1\\6\end{bmatrix}\)

The balanced equation is:

\(2\ \text{Na}_3\text{P}\text{O}_4+3\ \text{Ba}(\text{N}\text{O}_3)_2 \rightarrow \text{Ba}_3(\text{P}\text{O}_4)_2+6\ \text{Na}\text{N}\text{O}_3\)

12.

Node analysis gives:

\({\bf A:}\ \ x_1=40+x_3+x_4\)

\({\bf B:}\ \ 200=x_1+x_2\)

\({\bf C:}\ \ x_2+x_3=100+x_5\)

\({\bf D:}\ \ x_4+x_5=60\)

\({\bf \infty:}\ \ 200=40+100+60\)

Rearrange so it’s easier to write in matrix form:

\(x_1-x_3-x_4=40\)

\(x_1+x_2=200\)

\(x_2+x_3-x_5=100\)

\(x_4+x_5=60\)

The augmented matrix is:

\(\begin{bmatrix} 1&0&-1&-1&0&40\\ 1&1&0&0&0&200\\ 0&1&1&0&-1&100\\ 0&0&0&1&1&60 \end{bmatrix}\)

which reduces to:

\(\begin{bmatrix} 1&0&-1&0&1&100\\ 0&1&1&0&-1&100\\ 0&0&0&1&1&60\\ 0&0&0&0&0&0 \end{bmatrix}\)

a) So, the general traffic pattern is:

\(x_1=100+x_3-x_5\)

\(x_2=100-x_3+x_5\)

\(x_3\) is free

\(x_4=60-x_5\)

\(x_5\) is free

b) For \(x_4=0\) we must have \(x_5=60\). This gives:

\(x_1=40+x_3\)

\(x_2=160-x_3\)

\(x_3\) is free

\(x_4=0\)

\(x_5=60\)

c) The variables represent number of cars, so the smallest \(x_1\) could be is if \(x_3=0\) and hence \(x_1=40\).
