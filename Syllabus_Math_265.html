Course description

An introduction to the fundamental algebraic, logical, and combinatorial concepts of mathematics that provides a foundation for the study of computer science. Prerequisites: Student has completed all of the following course(s): MATH 221 - Calculus I with grade greater than or equal to C- (Undergraduate Grading Scheme).

Course Learning Objectives

Students who successfully complete this course will be expected to:

  • demonstrate an understanding of functions in multiple ways including graphical representations, algebraic representations, and verbal descriptions
  • demonstrate understanding of basic logic, including quantifiers, negations
  • understand basic proof structure and be able to correctly write short proofs
  • demonstrate a basic understanding of sets and functions
  • be able to apply basic counting principles and compute basic probabilities
  • know basic properties of graphs and trees, including some traversal algorithms.

Some specific topics include:

  1. Basic logic
    • Truth tables.
    • Mathematical deduction and proof.
  2. Basic set theory
    • Set operations – union, intersection, complement. Representation using Venn diagrams. De Morgan’s laws.
    • Subsets, power sets, examples.
  3. Relations and functions
    • Equivalence relations, congruence, partitions.
    • Domain, range, co-domain, injectivity, surjectivity. Modular arithmetic.
  4. Combinatorics
    • Fundamental principle of counting, permutations, combinations.
    • Binomial theorem, Pascal’s triangle.
  5. Graph theory
    • Basic definitions.
    • Planar graphs, Hamiltonian and Euler paths, graph coloring.
    • Directed and undirected graphs.
    • Minimum spanning trees.
    • Computational complexity.
  6. Induction
    • Principle of mathematical induction, strong induction.
  7. Recursion
  8. Recursive algorithms.
  9. Recursive data structures.
